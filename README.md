# NegoIT POS System

NegoIT POS System is a comprehensive point-of-sale solution built with PHP. This system handles billing, inventory, customer management, and integrates with QuickBooks for accounting.

## 📋 Features

- POS Billing (Bill 1 & Bill 2)
- Customer Management
- Order Processing
- QuickBooks Integration
- SMS Gateway Integration
- Department Management
- Payment Processing
- Returns Management
- Real-time Health Monitoring

## 🛠️ System Requirements

- **PHP:** 8.0, 8.1, 8.2, 8.3, or 8.4
- **MySQL:** 5.7 or higher
- **Web Server:** Apache/Nginx
- **OS:** Linux/Windows/MacOS
